# エリアページ コンテンツ戦略メモ

## 目的

46エリアの個別ページで「地域名 + ピアノ教室」「地域名 + 大人 + ピアノ」「地域名 + 出張レッスン」の検索意図を取りに行き、問い合わせ前の不安を解消する。

現在の主ターゲットは、大人の初心者・再開者・趣味としてピアノを続けたい方。子ども向け・保護者向けの訴求は、ページ本文の主軸にしない。

---

## ページ内で必ず扱う不安

1. 楽譜が読めなくても始められるか
2. ブランクがあっても再開できるか
3. 電子ピアノでも受講できるか
4. マンションでも出張レッスンが可能か
5. 自宅が難しい場合にスタジオ利用できるか
6. 仕事や家事で毎週固定が難しい場合の相談可否
7. 出張費・交通費が料金に含まれるか
8. 体験前にLINEで相談できるか

## 内部リンク方針

- 同じグループの近隣エリアを優先して表示する
- 中央区・江東区の主要エリアからトップ、`/lesson`、`/contact` へ自然に流す
- NEWS記事から関連エリアページへリンクする
- エリア一覧 `/area` は全エリアへのハブとして維持する

## Search Consoleで追う指標

週次ではなく、月次で確認する。

| 指標 | 見る理由 |
| --- | --- |
| 地域名 + ピアノ教室の表示回数 | エリアページが検索対象に入っているか |
| 地域名 + 大人 + ピアノの順位 | ターゲット訴求が拾えているか |
| 地域名 + 出張ピアノの順位 | 出張型の差別化が効いているか |
| CTR | title/descriptionが検索意図に合っているか |
| `/contact?from=area-{slug}` の送信数 | エリア別CV |

## 改善判断

| 状態 | 次の施策 |
| --- | --- |
| 表示回数あり、順位11〜30位 | FAQや本文に該当クエリを自然に追記 |
| 表示回数あり、CTR低い | title/meta descriptionを改善 |
| 表示回数なし | 内部リンク、本文量、関連NEWS記事を追加 |
| 問い合わせあり | 近隣エリアにも同じ不安解消を展開 |

---

## 注意点

- 地域名だけを置換した薄い文章にしない
- 子ども向け・保護者向け訴求を主見出しにしない
- 「おすすめ」「人気」など比較サイト的な表現を過度に使わない
- 実際に対応できない時間帯・エリアを強く書きすぎない
- FAQPage構造化データは補助と考え、本文そのものの分かりやすさを優先する
