# 集客キーワードリスト

髙橋遊月ピアノ教室のSEOで優先するキーワードを、大人向けの出張ピアノレッスンに合わせて整理します。

現在の主ターゲットは「大人の初心者・再開者・趣味としてピアノを続けたい方」です。子ども向け・保護者向けキーワードは、問い合わせ実績や方針変更がない限り低優先度にします。

---

## 優先ターゲット

| 区分 | 想定読者 | 主な検索意図 |
| --- | --- | --- |
| 大人の初心者 | 楽譜が読めない、楽器選びから不安 | 始め方・不安解消・体験相談 |
| 大人の再開者 | 子どもの頃以来、ブランクが長い | 戻し方・練習方法・無理ない頻度 |
| 忙しい社会人 | 仕事後・休日に趣味として続けたい | 通学不要・出張・日程調整 |
| 50代・60代 | 定年後や自分時間の趣味を探す | シニア対応・ゆっくり進める安心感 |
| 自宅/スタジオ検討者 | 電子ピアノ、自宅環境、スタジオ利用が気になる | 受講条件・料金・出張範囲 |

## 低優先度・原則狙わないキーワード

今のサイト方針では、以下は優先度を下げます。記事化する場合も、大人向け導線を邪魔しない範囲にします。

| キーワード例 | 理由 |
| --- | --- |
| 子ども ピアノ教室 東京 | 現在の主訴求とズレる |
| 子ども ピアノ いつから | 保護者向けの一般情報になりやすい |
| ピアノ 練習 嫌がる 子ども | 大人向けCVに遠い |
| ピアノ コンクール 小学生 | 教室の主軸とズレる |
| リトミック / 幼児音楽教室 | サービス内容とズレる |

## キーワード配置ルール

1. メインキーワードはタイトル前半、H1、冒頭100文字以内に入れる
2. 地域ページでは「地域名 + ピアノ教室」をtitleとH1近辺に入れる
3. 記事では「悩み語句 + 大人」を自然に入れる
4. 同じ語句を詰め込まず、「出張」「訪問」「自宅」「スタジオ」「マンツーマン」で言い換える
5. すべての記事から `/contact`、`/lesson`、関連エリアページへ内部リンクする
