# SEOライティングガイド

NEWS記事を書く際のSEOライティングルールをまとめています。
検索エンジンに評価され、かつ読者にとっても読みやすい記事を目指しましょう。

---

## 1. キーワード配置のルール

### 必ずキーワードを含める場所

| 配置場所 | 重要度 | 注意点 |
|---------|-------|-------|
| 記事タイトル（title） | ★★★ | 前半にメインKWを配置 |
| メタディスクリプション | ★★★ | 80〜120文字で記事要約 |
| H2見出し（最低1つ） | ★★★ | 不自然にならない範囲で |
| 記事の冒頭100文字以内 | ★★ | 導入文で自然に使う |
| 画像のalt属性 | ★★ | 画像の説明として自然に |
| URL（スラッグ） | ★ | ローマ字で短く |

### キーワード密度の目安

- メインキーワード: 記事全体の**1〜2%**（3,000文字なら5〜10回程度）
- **不自然な繰り返しはNG**（Googleにスパム判定される）
- 同義語・関連語で言い換えを活用する

### 例: メインKW「ピアノ 初心者 練習方法」の場合

```
✅ 良い例:
「ピアノ初心者の方に向けて、効果的な練習方法をご紹介します」
「初心者がピアノを練習する際に大切なポイントは...」
「ピアノの練習で初心者が気をつけるべきことは...」

❌ 悪い例:
「ピアノ初心者練習方法をお探しの方にピアノ初心者練習方法を解説」
```

## 2. 記事構成のルール

### 基本構成

```
タイトル（H1）← 1記事に1つだけ
  │
  ├── 導入文（リード文）
  │
  ├── H2: メインテーマ1
  │     ├── H3: サブトピック
  │     └── H3: サブトピック
  │
  ├── H2: メインテーマ2
  │     ├── H3: サブトピック
  │     └── H3: サブトピック
  │
  ├── H2: FAQ（よくある質問）
  │
  └── H2: まとめ + CTA
```

### 見出しのルール

- **H2は3〜6個**が適切（多すぎると散漫、少なすぎると薄い）
- **H3はH2の下に2〜4個**
- 見出しだけ読んで記事の内容がわかるようにする
- 見出しにもキーワードを自然に含める
- H2 → H3 → H4 の順番を飛ばさない

---

## 3. 文字数の目安

| 記事タイプ | 推奨文字数 | 備考 |
|-----------|----------|------|
| ピアノ練習のコツ | 2,000〜3,000文字 | 詳しく解説 |
| エリア紹介 | 2,000〜3,000文字 | 地域情報を充実 |
| 生徒の声 | 1,500〜2,500文字 | エピソード中心 |
| レッスンレポート | 1,000〜2,000文字 | 気軽な更新 |

**最低1,500文字**を目安にしてください。1,000文字未満の記事は検索上位に入りにくい傾向があります。

---

## 4. 読みやすさのルール

### 文章の書き方

- **一文は60文字以内**を目安に（長すぎると読みづらい）
- **一段落は3〜4文**で改行する
- 「です・ます」調で統一（敬体）
- 専門用語を使う場合は簡単な説明を添える

### 視覚的な読みやすさ

- **箇条書き・リスト**を活用する
- **太字**で重要ポイントを強調する
- 表（テーブル）で比較情報をまとめる
- 3〜4段落ごとに**画像を挿入**する
- 適切な余白（空行）を入れる

### 避けるべき表現

- 「〜と思います」の多用 → 断定できる場合は言い切る
- 「〜になります」→ 「〜です」
- 「まず最初に」→ 「まず」（重複表現）
- 過度な敬語 → 適度にくだけた表現で親しみやすく

## 5. 画像の最適化

### alt属性（代替テキスト）の書き方

画像のalt属性はSEOに影響するため、適切に設定してください。

```
✅ 良い例:
alt="ピアノを弾く子どもの手元"
alt="江東区の出張ピアノレッスンの様子"
alt="初心者向けピアノ教材バイエルの表紙"

❌ 悪い例:
alt="画像1"
alt=""（空のまま）
alt="ピアノ ピアノ教室 東京 レッスン"（キーワード詰め込み）
```

### 画像のルール

- ファイル名もわかりやすく: `piano-lesson-koto-chuo.jpg`
- ファイルサイズは**500KB以下**に圧縮
- 推奨サイズ: 幅1200px程度
- アイキャッチ画像は**1200×630px**（OGP対応）

## 6. 内部リンクのルール

### 基本方針

- 1記事に**3〜5つの内部リンク**を入れる
- **体験レッスン申込みページへのリンクは必須**
- 関連する過去記事にもリンクを貼る
- アンカーテキスト（リンク文字）にキーワードを含める

### アンカーテキストの例

```
✅ 良い例:
「詳しくは[ピアノ教室の料金ページ](リンク)をご覧ください」
「[江東区・中央区の出張ピアノレッスン](リンク)について詳しくはこちら」

❌ 悪い例:
「詳しくは[こちら](リンク)」（キーワードが含まれていない）
```

## 7. CTA（行動喚起）の配置

### CTAを入れる場所

1. **記事中盤**（H2の2つ目の後あたり）: さりげなく
2. **記事末尾**（まとめの後）: しっかりと

### CTAの文例

```markdown
> 🎹 髙橋遊月ピアノ教室では無料体験レッスンを受付中です。
> 「もっと詳しく知りたい」「一度試してみたい」という方は、
> お気軽にお問い合わせください。
> → [無料体験レッスンに申し込む](リンク)
```

### CTAのコツ

- 記事の内容と関連づける（「この練習法をもっと知りたい方は...」）
- 押し売り感を出さない
- ボタン風のリンクでクリック率UP

---

## 8. メタディスクリプションの書き方

### ルール

- **80〜120文字**で記事の内容を要約
- メインキーワードを含める
- 「読みたい」と思わせる内容
- 記事ごとにユニーク（使い回さない）

### テンプレート

```
[テーマ]について[誰が]解説。[具体的な内容]をご紹介します。[教室名]は[場所]で[CTA]。
```

### 例

```
ピアノ初心者向けの練習方法を現役講師が解説。上達のコツをステップ別にご紹介します。
髙橋遊月ピアノ教室は東京都江東区・中央区で無料体験レッスン受付中。（82文字）
```

## 9. 公開前チェックリスト

記事を公開する前に以下を確認:

- [ ] タイトルにメインキーワードが含まれているか（前半に）
- [ ] メタディスクリプションを設定したか（80〜120文字）
- [ ] H2見出しにキーワードが含まれているか（最低1つ）
- [ ] 記事冒頭100文字以内にキーワードがあるか
- [ ] 文字数は最低1,500文字以上か
- [ ] 画像のalt属性を設定したか
- [ ] 内部リンクを3つ以上入れたか
- [ ] 体験レッスンへのCTAを入れたか（中盤+末尾）
- [ ] 誤字脱字がないか
- [ ] スマホで読みやすいか（段落が長すぎないか）
