# 記事構成の指示
以下のルールで構成を考え、記事を書いてください:

1. **導入文（150〜300文字）**: 読者の悩みに共感 → 記事の内容を予告
2. **H2見出しを4〜6個**: 盛り込みたい内容を論理的な順序で構成
3. **各H2の下にH3を2〜3個**: 具体的なサブトピックを展開
4. **H2: よくある質問（FAQ）**: 3〜5問のQ&A形式
5. **H2: まとめ + CTA**: 要点の箇条書き → 以下のCTAを必ず含める:

---
**無料体験レッスン受付中！**
髙橋遊月ピアノ教室では、一人ひとりに合わせた指導を行っています。
ピアノ選びのご相談もお気軽にどうぞ。
📍 東京都江東区・中央区を中心に出張レッスン対応 🎹 初回体験レッスン無料
→ 体験レッスンのお申込みはこちら
---

# SEOルール（必ず守ること）
- タイトルは30〜60文字。メインキーワードを前半に配置
- メタディスクリプションを80〜120文字で作成
- H2見出しの最低1つにメインキーワードを含める
- 記事冒頭100文字以内にメインキーワードを含める
- 記事中盤にもCTAを自然に挿入する
- 合計文字数は2,000〜3,000文字
- 「です・ます」調で統一
- 一文は60文字以内、一段落は3〜4文
- 箇条書き・太字・表を適宜使用して読みやすく
- 専門用語には簡単な説明を添える

# 出力形式
以下の形式で出力してください:

**【タイトル】**
（タイトルをここに）

**【メタディスクリプション】**
（80〜120文字でここに）

**【本文】**
（Markdown形式で記事本文をここに）
