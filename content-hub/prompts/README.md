# AI記事生成プロンプト 使い方ガイド

生成AI（ChatGPT / Claude / Gemini など）にコピペするだけで、
SEO最適化された記事を一発で作成できるプロンプト集です。

---

## 使い方（3ステップ）

### Step 1: プロンプトを選ぶ

| やりたいこと | 使うファイル |
|------------|------------|
| ピアノ練習のコツ記事を書く | `piano-tips-prompt.md` |
| エリア紹介記事を書く | `area-guide-prompt.md` |
| 生徒の声・体験談を書く | `student-voice-prompt.md` |
| レッスンレポートを書く | `lesson-report-prompt.md` |
| どのカテゴリにも当てはまらない記事 | `universal-prompt.md` |
| タイトル案をたくさん出したい | `title-generator-prompt.md` |
| メタディスクリプションだけ作りたい | `meta-description-prompt.md` |
| 既存記事をSEOリライトしたい | `rewrite-prompt.md` |

### Step 2: `{{変数}}` を埋める

各プロンプト内の `{{二重波括弧}}` 部分を、実際の内容に置き換えます。

```
例:
{{メインキーワード}} → ピアノ 初心者 練習方法
{{ターゲット読者}}   → ピアノを始めたばかりの大人の初心者
{{エリア名}}        → 東京都江東区
```

### Step 3: AIにコピペ → 出力をmicroCMSに入稿

1. 変数を埋めたプロンプト全体をコピー
2. ChatGPT / Claude などに貼り付けて送信
3. 出力された記事を確認・微調整
4. microCMSに入稿

---

## プロンプト一覧

### 記事生成（カテゴリ別）
- **`piano-tips-prompt.md`** — ピアノ練習のコツ記事をフル生成
- **`area-guide-prompt.md`** — エリア紹介記事をフル生成
- **`student-voice-prompt.md`** — 生徒の声記事をフル生成（素材を入力）
- **`lesson-report-prompt.md`** — レッスンレポートをフル生成（素材を入力）
- **`universal-prompt.md`** — カテゴリ不問の汎用記事生成

### 部分生成・ユーティリティ
- **`title-generator-prompt.md`** — タイトル案を10個以上一括生成
- **`meta-description-prompt.md`** — メタディスクリプションを5パターン生成
- **`rewrite-prompt.md`** — 既存記事のSEOリライト

---

## Tips

- **出力が途中で切れたら**: 「続きを書いてください」と送ればOK
- **もっと長くしたい**: プロンプト内の文字数指定を増やす
- **トーンを変えたい**: プロンプト末尾に「もう少しカジュアルに」等を追記
- **画像の指示が欲しい**: 出力後に「各セクションに適した画像の提案もください」と追加で聞く
- **複数記事を量産**: 変数だけ変えて繰り返しコピペすれば大量生成可能
