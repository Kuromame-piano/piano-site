# メタディスクリプション生成プロンプト

記事のメタディスクリプションだけを複数パターン生成したい時に使います。
下の `{{変数}}` を埋めて、全体をコピペしてください。

---

## ▼ ここから下を全部コピー ▼

```
以下の記事情報をもとに、メタディスクリプション（検索結果に表示される要約文）を5パターン生成してください。

# 記事情報
- 記事タイトル: {{記事タイトル（例: 【初心者向け】ピアノの練習方法を基礎から解説）}}
- メインキーワード: {{メインキーワード（例: ピアノ 初心者 練習方法）}}
- 記事の要約: {{記事の内容を2〜3文で（例: ピアノ初心者向けに効率的な練習方法を解説。基礎練習のステップ、よくある間違い、講師おすすめの練習メニューを紹介。）}}
- 教室名: 髙橋遊月ピアノ教室（東京都江東区・中央区・出張レッスン・体験無料）

# ルール
- 各パターン80〜120文字
- メインキーワードを必ず含める
- 5パターンそれぞれ異なるアプローチで:
  1. 情報提供型（「〇〇を解説します」）
  2. 悩み共感型（「〇〇でお悩みの方へ」）
  3. メリット訴求型（「〇〇で上達を実感」）
  4. CTA含有型（「体験レッスン受付中」を含む）
  5. 数字/具体性型（数字や具体的な成果を含む）

# 出力形式
| # | タイプ | メタディスクリプション | 文字数 |
|---|-------|---------------------|-------|
| 1 | 情報提供型 | ... | 95文字 |

最後に「おすすめ」を1つ選んで理由を添えてください。
```

```
- 記事の要約: 大人からピアノを始めることに不安を感じている方へ、現役講師が本音で回答。始めるのに遅すぎることはない理由、大人ならではの上達のコツ、おすすめの練習方法を解説。
```
