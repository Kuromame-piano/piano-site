## 🎨 背景画像の分析

### 色味・トーン
| 要素 | 分析 |
|------|------|
| **主調色** | ベージュ〜クリーム系（#E8DFD4 〜 #F5EFE7） |
| **ピアノ木目** | ウォールナット系ブラウン（#8B6F4E 〜 #A68B5B） |
| **ピアノ鍵盤** | アイボリー（白鍵）、ダークグレー〜ブラック（黒鍵） |
| **オーバーレイ** | 左側にソフトフォグ効果（クリーム〜ホワイトのグラデーション） |
| **全体印象** | 落ち着いた暖色系、柔らかな光、上品で高級感のある雰囲気 |

### テキスト配置エリアの特徴
- 左〜中央に白いミスト状のオーバーレイが存在
- 読みやすさを確保しやすいエリアだが、コントラストがやや弱い
- 背景のブラウン〜ベージュ系と調和する色選びが重要

---

## 🔍 現状の課題

| 課題 | 詳細 |
|------|------|
| **統一感の欠如** | 見出しにSerif、説明文にSans-serifを使用しており、フォントファミリーの混在が「まとまり」を損なっている |
| **視認性** | 現在の色（#544A40）は背景と同系色のため、コントラストがやや弱い |
| **階層感の不明瞭** | 各テキスト層の差別化が不十分で、情報の優先度が伝わりにくい |
| **上品さの不足** | 文字間隔・行間・ウェイトのバランスにより、もう少し洗練された印象を与える余地がある |

---

## ✨ デザインコンセプト

### 目指す雰囲気
> **「まとまりよく、上品で、大人っぽい」**

### デザインの方向性
1. **統一されたフォントファミリー** — 明朝体で統一し、クラシカルで上品な印象を強化
2. **階層構造の明確化** — サイズ・ウェイト・色でテキストの優先度を視覚的に表現
3. **背景との調和** — ブラウン系をベースに、視認性を高めるアクセントカラーを追加
4. **余白と間隔の最適化** — ゆとりある文字間隔・行間で「大人の余裕」を演出

## 📐 CSSパラメータ提案

### 第1階層：トップラベル
> 「江東区・清澄白河エリアの出張ピアノレッスン」

| プロパティ | 現状値 | 提案値 | 理由 |
|-----------|--------|--------|------|
| `font-family` | `Shippori Mincho` | `"Shippori Mincho", "Noto Serif JP", serif` | フォールバック追加で安定表示 |
| `font-size` | `text-xl (20px) / md:text-2xl (24px)` | `text-sm (14px) / md:text-base (16px)` | **小さめに抑え、メイン見出しを引き立てる**。控えめなサイズで上品さを演出 |
| `font-weight` | `700 (bold)` | `500 (medium)` | 太すぎると主張が強くなるため、**ミディアムウェイトで品格を維持** |
| `letter-spacing` | `tracking-[0.2em]` | `tracking-[0.25em]` | **ゆったりした文字間隔で高級感を強調**。ラベルとしての役割を明確化 |
| `line-height` | 未設定 | `leading-relaxed (1.625)` | 余裕のある行間でエレガントさを追加 |
| `color` | `#544A40` | `#A88B58` | **ゴールドブラウンで、背景のピアノ木目と調和しつつアクセントに**。メイン見出しとの差別化 |

**提案CSS（Tailwind）:**
```css
class="font-serif text-sm md:text-base font-medium tracking-[0.25em] leading-relaxed text-[#A88B58] drop-shadow-sm"
```

### 第2階層：メイン見出し
> 「日常に彩りを添える、心豊かなピアノの時間」

| プロパティ | 現状値 | 提案値 | 理由 |
|-----------|--------|--------|------|
| `font-family` | `Shippori Mincho` | `"Shippori Mincho", "Noto Serif JP", serif` | 継続使用、フォールバック追加 |
| `font-size` | `text-2xl (24px) / md:text-5xl (48px)` | `text-2xl (24px) / md:text-4xl (36px) / lg:text-[42px]` | **やや控えめにして余裕のある印象に**。大きすぎると圧迫感が出る |
| `font-weight` | `700 (bold)` | `600 (semibold)` | **セミボールドで力強さを残しつつ、繊細さも表現** |
| `letter-spacing` | 未設定（ブラウザデフォルト） | `tracking-[0.08em]` | **程よい間隔で読みやすさと上品さを両立** |
| `line-height` | 未設定 | `leading-snug (1.375)` | 2行にまたがる見出しのバランスを最適化 |
| `color` | `#544A40` | `#3D362F` | **より深いブラウンで、視認性とコントラストを強化**。背景の淡いベージュに対して程よい重み |

**追加効果:**
- `drop-shadow-md` — 軽いシャドウで文字を浮かせ、背景との分離を強化
- `text-shadow: 0 2px 8px rgba(252, 250, 247, 0.6)` — 背景の明るい部分と馴染ませるオプション

**提案CSS（Tailwind）:**
```css
class="font-serif text-2xl md:text-4xl lg:text-[42px] font-semibold tracking-[0.08em] leading-snug text-[#3D362F] drop-shadow-md"
```

### 第3階層：説明文
> 「講師がご自宅へ伺い、あなたの大切なピアノで、あなただけの音色を一緒に奏でます。」

| プロパティ | 現状値 | 提案値 | 理由 |
|-----------|--------|--------|------|
| `font-family` | `ui-sans-serif` | `"Shippori Mincho", "Noto Serif JP", serif` | **明朝体で統一し、全体の一体感を確立**。Sans-serifとの混在をなくす |
| `font-size` | `text-base (16px) / md:text-lg (18px)` | `text-sm (14px) / md:text-base (16px)` | **控えめなサイズで、見出しとの階層差を明確化** |
| `font-weight` | `400 (normal)` | `400 (normal)` | 変更なし。サポート役として軽やかに |
| `letter-spacing` | 未設定 | `tracking-[0.06em]` | **日本語テキストに適度な余裕を持たせ、読みやすさ向上** |
| `line-height` | `leading-loose (2.0)` | `leading-relaxed (1.625)` | **やや詰めて、よりコンパクトに**。2.0は広すぎてまとまりが欠ける |
| `color` | `#544A40` | `#5A4F44` | **メイン見出しより明るめのブラウンで、情報の優先度を表現** |

**提案CSS（Tailwind）:**
```css
class="font-serif text-sm md:text-base font-normal tracking-[0.06em] leading-relaxed text-[#5A4F44]"
```

### 色選定の根拠
1. **背景との調和**: すべてブラウン系で統一し、背景画像のウォームトーンと自然に融合
2. **視認性確保**: 背景のクリーム系（明度高）に対し、ダークブラウン（明度低）で十分なコントラスト
3. **階層表現**: 色の濃度・彩度で情報の重要度を視覚的に表現

## 💡 追加推奨事項

### 1. 背景オーバーレイの微調整
現在のグラデーション：
```css
bg-gradient-to-b from-[#FCFAF7]/20 to-[#FCFAF7]
```
推奨変更：
```css
bg-gradient-to-b from-[#FCFAF7]/30 via-[#FCFAF7]/50 to-[#FCFAF7]
```
**理由**: 中間地点のオーバーレイを強化し、テキストエリアの視認性をさらに向上

### 2. テキストシャドウの活用
メイン見出しに微細なシャドウを追加することで、背景との分離を強化しつつ、浮き上がるような立体感を演出：
```css
text-shadow: 0 1px 3px rgba(255, 255, 255, 0.3), 0 3px 12px rgba(0, 0, 0, 0.05);
```

### 3. レスポンシブ対応の強化
モバイル表示時のフォントサイズと行間をさらに細かく調整し、小画面でも読みやすさと美しさを維持
